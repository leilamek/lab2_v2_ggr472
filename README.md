# lab2_v2_ggr472
**Lab 2 GGR472: Etobicoke General Hospital Map**
2nd version of lab 2 repository for GGR472 course
Includes an html, css, and js file with comments. html is embedded with a map box map
displaying in red a polygon shape of the Etobicoke general hospital, and a red dot 
indicating the Humber College LRT station. These 2 data types (building-polygon and transit-point)
are related in terms of public-health, accessibility, and the ease in which individuals 
(with or without personal-vehicles) may access public health services, and the proximity of
service-providers to public transit hubs.
